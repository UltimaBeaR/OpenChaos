# Gameplay Changes — OpenChaos vs Original

Все изменения функциональности игры относительно **релизной PC-версии** Urban Chaos.
Здесь описываются изменения геймплея, управления, читов, механик — всё что влияет на игровой опыт.
Графические изменения (рендерер, эффекты) сюда не входят.

---

## Скорость игры

Скорость игры замедлена и приведена к реальному времени по игровому таймеру.

- Скорость физики в оригинале на ПК — 22 итерации в секунду.
- У нас — 20.

---

## Управление

### Геймпад (заменено)
- Убрана поддержка оригинальных PC-джойстиков (DirectInput).
- Добавлена полностью новая поддержка геймпадов: DualSense (нативный) и Xbox/generic (через SDL3).
- Раскладка кнопок своя — не повторяет 1:1 ни PC, ни PS1, ни Dreamcast версию.
- DualSense: LED подсветка, адаптивные триггеры, вибрация.

### Клавиатура и мышь — новая раскладка (заменено)
Старая раскладка с танковым управлением на стрелках полностью заменена на современную WASD + мышь. Переназначение клавиш вернётся в одной из следующих версий (после 1.0), уже на новой системе биндингов.

**Передвижение (на ноге):**

| Действие | Кнопка |
|----------|--------|
| Движение (аналоговое — диагонали клампятся до единичного вектора, как настоящий стик) | **W A S D** |
| Поворот камеры | **Мышь** (yaw + pitch) |
| Прыжок | **Space** |
| Удар рукой | **ЛКМ** |
| Удар ногой | **ПКМ** |
| Зум камеры / ход назад (hold) | **E** (стоя = зум; +WASD = ход назад) |
| Спринт (hold при беге) | **Left Shift** |
| Стелс-присед (hold) | **C** |
| Действие (взять / сесть в машину / обыск / арест / рычаги / ...) | **F** |
| Дизарм / последнее оружие | **СКМ** (клик колёсика) |
| Листать инвентарь | **колесо мыши** вверх/вниз |
| Прямой выбор оружия | **1 2 3 4** (пистолет / автомат / дробовик / граната), **Tab** (ближний бой) |
| Пауза | **Esc** |
| Консоль разработчика | **F9** |

**В машине:**

| Действие | Кнопка |
|----------|--------|
| Газ | **W** |
| Тормоз | **Space** |
| Задний ход (реверс) | **S** |
| Руль | **A / D** |
| Сирена (полицейская машина) | **E** |
| Выйти из машины | **F** |

**Старое управление стрелками (танковое):** клавиши ↑↓←→ больше ничего не делают на ноге. Поддержка танкового движения сохранена в коде закомментированной — в одной из будущих версий вернётся как опциональный тумблер в настройках.

**Прежние клавиши (Z = удар, X = пинок, Enter = инвентарь, A = aim hold, F5-F7/End/Del/PgDn = камера):** все убраны. Камеру теперь крутит мышь; удары — мышью; спринт — на Shift, стелс — на C, действие — на F; смена оружия — колесо / средняя кнопка / 1–4 / Tab (см. ниже).

**Навигация в меню на WASD (новое):** во всех меню (фронтенд, пауза, выбор миссии, win/lose-экраны, формы) перемещение по пунктам теперь работает не только стрелками ↑↓←→, но и WASD. Стрелки остаются — WASD добавлен как альтернатива. В полях ввода текста WASD печатают буквы как обычно, а не навигируют.

### Кнопка действия разделена на три (изменено)
Раньше всё висело на одной кнопке действия (геймпад Circle / клава Shift): она же спринт, она же присед-стелс, она же все интеракции — и игра сама решала что сделать по ситуации. Теперь это три отдельные кнопки:

| Действие | Геймпад | Клавиатура |
|----------|---------|------------|
| Спринт (hold при беге) | **Circle** | **Left Shift** |
| Стелс-присед (hold) | **Triangle** | **C** |
| Действие / интеракции (сесть-выйти из машины, поднять, обыск, арест, рычаги, грэпплинг и т.д.) | **Square** | **F** |

«Выбор нужного действия по ситуации» остался только на кнопке действия (Square/F) — спринт и стелс теперь простые и независимые. Арест переехал на кнопку действия (раньше срабатывал и при приседании). Присед при обыске трупа сохранился — он часть самой анимации обыска и включается сам.

### Смена оружия — новая схема (изменено)
Схема выбора оружия полностью переработана. Старый «умный авто-выбор лучшего оружия» (был на R3 / Tab когда безоружен) **убран**.

**Геймпад:**

| Кнопка | Действие |
|--------|----------|
| R3 (клик правого стика) | **Дизарм / последнее оружие.** В руках любое оружие (в т.ч. ближнее) → убрать (в кулаки). Пустые руки → достать последнее что было (или первое в списке, если последнего больше нет) |
| Зажать R3 + D-pad ↑/↓ | Листать инвентарь по списку (с заворотом low↔top) |
| D-pad ↑ | Пистолет |
| D-pad ← | Автомат (AK47) |
| D-pad → | Дробовик |
| D-pad ↑+→ (вместе) | Граната |
| D-pad ↓ | Ближний бой — бита ↔ нож (только то что есть; кулаки тут не выбираются) |

**Клавомышь:**

| Кнопка | Действие |
|--------|----------|
| Средняя кнопка мыши (клик колёсика) | Дизарм / последнее оружие (как R3) |
| Колесо вверх/вниз | Листать инвентарь по списку (с заворотом) |
| 1 / 2 / 3 / 4 | Пистолет / Автомат / Дробовик / Граната |
| Tab | Ближний бой (бита ↔ нож, что есть) |

Прямые кнопки (D-pad / 1–4 / Tab) **не дизармят** — дизарм только R3 / средняя кнопка. Редкое оружие (взрывчатка и т.п.) прямой кнопки не имеет — берётся прокруткой. При любом выборе всплывает панель инвентаря, выбранное подсвечивается и берётся в руки с анимацией.

**Пустое оружие больше не переключается само.** Раньше при попытке выстрела с пустым магазином игра автоматически доставала следующее оружие с патронами (а если ни у чего патронов нет — убирала в кулаки). Теперь огнестрел с пустым магазином просто сухо щёлкает и **остаётся в руках** — смену оружия делает игрок сам. (Гранаты: когда брошена последняя, предмет пропадает из инвентаря и руки становятся пустыми — это не изменилось.)

### L2 — перекаты и медленный шаг (новое)

L2 на геймпаде работает по-разному в зависимости от того, **тапнуть** его или **держать**. Триггер срабатывает у **самого низа** хода — нужно прожать почти до конца (как удар рукой на R2); лёгкое касание не считается.

**Перекат — быстрый тап L2.** Коротко нажал-отпустил L2, держа стик в любую сторону (кроме центра) → перекат туда, куда показывает стик в момент отпускания (с учётом камеры и текущего разворота перса). Куда стик — туда и перекат. Работает стоя, на ходу, на спринте и в бою.

**Медленный шаг — зажать L2 и держать.** Пока L2 удерживается со стиком в любую сторону → медленный камеро-ориентированный шаг (как обычный бег, но медленно и с анимацией шага).

**В бою.** Перекат делается либо крестиком (✕), либо тапом L2 — оба катят в направление стика, в любую сторону (не только влево/вправо).

**На клавиатуре** всё то же самое: WASD = стик, **Ctrl** = L2. Тап Ctrl + WASD → перекат в это направление (с учётом камеры; 8 направлений); зажать Ctrl + WASD → медленный шаг. Работает так же на ходу, на спринте и в бою. Ctrl — обычная клавиша, так что «срабатывание у самого низа» к ней не относится: нажал = включилось.

### L1 — зум и ход назад (новое)

L1 на геймпаде (на клавомыши — **E**) — мультитул: совмещает зум-с-кручением и ход назад. Что включится — зависит от того, стоит перс или двигается в момент нажатия.

**Зум — зажать L1 стоя на месте (стик в нейтрали).** Камера приближается к персонажу, мышь / правый стик крутят перса на месте (это единственный режим, где можно развернуть самого перса камерой). Из этого положения можно и стрелять.

**Ход назад — зажать L1 + стик.** Стик назад относительно перса → медленное пятение назад с танковым разворотом (перс плавно доворачивается следом за стиком, шагая спиной). Стик в другую сторону — обычное движение. «Назад» определяется относительно перса с учётом камеры: если перс смотрит мордой в камеру и игрок жмёт стик «вверх», это «назад для перса». Если L1 зажать сразу на ходу — попадаешь прямо в режим хода назад (зум — только со стойки). Из хода назад зум сам не вернётся: нужно отпустить L1 и зажать заново.

**Сальто назад.** Во время хода назад нажать прыжок (✕) → сальто назад (если место позволяет; иначе обычный прыжок).

**Камера.** Пока идёт ход назад (L1 / E зажат со стиком), автокамера не доворачивается за спину перса — чтобы вид не «перекидывало», когда пятишься.

**Убрано:** левый стик / WASD больше не дублируют кручение камеры в зуме (раньше крутили перса наравне с мышью/правым стиком). Теперь любое касание стика в зуме переводит в ход назад.

### Отмена/назад в UI — Circle вместо Triangle (изменено)
Во всех меню и UI-диалогах кнопка «назад / закрыть / отмена» на геймпаде теперь **Circle** (◯), а не Triangle (△). Это современная PS-конвенция: Cross подтверждает, Circle отменяет. Касается всего: pause-меню (закрытие), фронтенд-меню (back-out из подменю), внутриигровые формы (диалог «Leave the area?»), пропуск катсцен/аутро (Circle как доп. кнопка к Cross/Start).

Triangle (△) на геймпаде больше не задействован в UI вообще. Из игровых функций за ним остаётся только мигалка в полицейской машине.

### Экран «Mechanics» в паузе; настройка управления убрана (новое / убрано)
В меню паузы добавлен пункт **Mechanics** — справка по управлению и механикам игры. Подстраивается под активное устройство (клавиатура+мышь / Xbox / DualSense): показывает карту всех используемых кнопок (первый пункт **Controls**) и разделы по режимам — передвижение, ближний бой, оружие и стрельба, вождение, прочее.

Оригинальные пункты настройки управления (переназначение клавиш, выбор схемы) **убраны** — изменить раскладку больше нельзя, биндинги фиксированы.

## Читы

### Геймпад-читы (новое)
Комбинация: **Select + L1 + L2 + стрелка** (DualSense: Share+L1+L2, Xbox: Back+LB+LT).
Портированы из Dreamcast-версии с изменённой комбинацией кнопок.

| Стрелка | Эффект | Сообщение на экране |
|---------|--------|---------------------|
| Up | Бессмертие (toggle) | "I am immortal, I have inside me blood of kings." / "I'm just a mortal after all." |
| Down | Полное здоровье (1000 HP) | "My wings are as a shield of steel." |
| Left | Спавн оружия вокруг игрока (AK47, дробовик, пистолет, 3 гранаты) | "We need guns. Lots of guns." |
| Right | Максимум патронов (240 всех типов) | "Well, to tell you the truth, in all this excitement, I've kinda lost track myself." |

Отличия от Dreamcast-оригинала:
- Другая комбинация кнопок (было Start+LT+RT, стало Select+L1+L2)
- Добавлен отдельный чит на бессмертие (в оригинале DC его не было через комбо)

### Чит-команды через консоль F9 (новое)
На клавиатуре доступны те же четыре чита, что и на геймпаде — вводом команды в F9-консоли. Команды — фрагменты из тех же фильмов, что и сообщения чита (Highlander / Matrix / Dirty Harry), для удобства запоминания.

| Команда | Эффект |
|---------|--------|
| `bloodofkings` | Бессмертие (toggle) |
| `shieldofsteel` | Полное здоровье (1000 HP) |
| `weneedguns` | Спавн оружия вокруг игрока |
| `losttrack` | Максимум патронов |

Команды работают без активации `bangunsnotgames` — это полноценные геймплей-читы, не debug-инструменты. Эффект и сообщение на экране — те же, что у геймпад-чита.

### Пунктиры прицела NPC (убрано из обычного режима)
- В релизной PC-версии при прицеливании NPC из огнестрельного оружия рисовались серые пунктирные линии от стрелка к цели. Это отладочная визуализация, оставленная в релизе (на PS1 её нет).
- Теперь эти линии видны только в debug overlay (F9 → bangunsnotgames → Ctrl).

### Debug overlay (изменено)
- **Было (оригинал):** Ctrl нужно зажимать и держать чтобы видеть отладочную информацию.
- **Стало:** Ctrl работает как toggle — нажал раз, отладка показывается; нажал ещё раз — скрывается. Держать не нужно.
- Debug mode по-прежнему включается через F9 → `bangunsnotgames`.
- Shift+F12 по-прежнему переключает cheat overlay (FPS, координаты).

## Транспорт

### Газ / тормоз / задний ход — раздельные кнопки (изменено)
Раньше тормоз и задний ход были на одной кнопке: на скорости — тормоз,
на малой скорости — реверс. Теперь это **три отдельных управления**:
- **Газ** — W / R2.
- **Тормоз** — Space / L1 (на скорости тормозит с заносом; назад не едет).
- **Задний ход** — S / L2.

Приоритет при одновременном нажатии: тормоз > задний ход > газ (раньше газ
перебивал тормоз — нельзя было тормозить, не отпустив газ).

Задний ход и газ работают зеркально: если жмёшь встречное направление на
ходу — машина сперва плавно тормозит до остановки, затем (если держать
кнопку ~0.25с) сама едет в нужную сторону. Отпустил и нажал снова — едет
сразу (на нуле скорости). Торможение при этом плавное на высокой скорости
и более резкое у самой остановки (как настоящая машина).

Левый стик на геймпаде управляет **только рулём** — вверх/вниз больше не
газ/тормоз (раньше из-за лёгкого смещения стика машина непроизвольно
тормозила в крутом повороте).

### Выйти из машины можно на малом ходу (изменено)
Раньше выйти можно было практически только при полной остановке. Теперь
выход срабатывает, пока машина ещё медленно катится — не нужно ждать
точного нуля.

### Машину больше не срывает в занос на поворотах (изменено)
В оригинале машину могло сорвать в занос при повороте, но срабатывало это
нестабильно — иногда на ровном месте, иногда не срабатывало даже на крутом
вираже. «Поворотный» занос убран полностью (и у игрока, и у ИИ): повороты
теперь предсказуемые. Намеренный занос от резкого торможения на скорости
остался.

### Можно залезть на обычную полицейскую машину (новое)
В оригинале на стандартную полицейскую машину нельзя было залезть — у неё была сломана физика крыши, персонаж проваливался либо отбрасывался обратно. Полицейский «внедорожник» (sedan-вариант) и все остальные машины работали нормально. Теперь обычная полицейская машина ведёт себя как все остальные — на капот и крышу можно спокойно запрыгнуть.

Побочно исправилось ещё две вещи:
- На Urban Shakedown (участок полиции) больше не телепортирует сквозь припаркованные машины при беге в борт на уровне зеркала.
- Камера, когда упирается в полицейскую машину, теперь корректно облизывает крышу, а не «проваливается» в неё.

## Стрельба

### Непрерывная автоматная стрельба на ходу
В оригинале при зажатом триггере стрельбы из M16:
- Переход из положения **стоя → бег/шаг** обрывал автоматную очередь. Чтобы продолжить стрелять, нужно было отпустить и заново зажать триггер.
- В режиме **шага** стрельба не работала вообще (ни одно оружие).

Теперь:
- При зажатом триггере **автоматная очередь не прерывается** при любых переходах стоя ↔ бег ↔ шаг.
- В режиме **шага стрельба работает** для всех видов оружия (пистолет, дробовик, автомат), включая непрерывную очередь из автомата.

Неясно, был ли это баг релиза или задуманное поведение.


## Рукопашный бой

Изменения боевой системы. Всё ниже — только для игрока; поведение ИИ
не изменено.

### Комбо
- Комбо больше не прерывается из-за тайминга нажатий (раньше следующий
  удар нужно было нажать в короткое временное окно после завершения
  анимации удара).
- У ударов в цепочке есть шанс не сработать; чем дальше удар в комбо, тем
  ниже его шанс.
- Есть защита от неудач: если удар не сработал, его шанс растёт, пока он
  не пройдёт.

### Захваты
- Бросок из захвата срабатывает не всегда (примерно в 60% случаев); при
  несрабатывании вместо броска — отталкивание врага. Повторные попытки
  повышают шанс.
- Оглушение захватом со спины не срабатывает на идущем или бегущем враге —
  только на стоящем.

### Откаты действий
Для ряда действий добавлен откат — повторное действие невозможно, пока
откат не пройдёт: подкат/подсечка ~5 с, арест ~12 с, захват ~7 с,
уклонение ~2 с, перекат/флип ~2 с. Попытка ареста во время отката —
сообщение «FAILED».

### Уклонение от удара
- Одно уклонение на одно нажатие «назад» (раньше при удержании «назад»
  уклонение держалось постоянно).
- Позу уклонения нельзя удерживать бесконечно — примерно через 1 с Дарси
  завершает уклонение сама.

### Перемещение в боевой стойке
Исправлены баги в перемещении: больше нельзя забивать врагов через баги
ввода и нельзя двигаться неестественно быстро.

### Добивание лежачего
Пинок по лежащему (в нокауте) врагу срабатывает с большего расстояния и
без точного направления (раньше требовалась точная позиция).

### Смена цели
- Кнопка смены цели выбирает врага по направлению взгляда камеры (раньше —
  в произвольном порядке). Среди тех, кто примерно на линии взгляда,
  выбирается ближайший к Дарси.
- Если повернуть камеру и нажать — выбирается тот, на кого направлен
  взгляд; если камеру не трогать — перебор по кругу.

### Авто-наведение при ударе
Перед каждым ударом (рукой и ногой, кроме боковых/задних ударов) Дарси
сама перенацеливается на ближайшего врага в направлении камеры — нажимать
кнопку смены цели перед каждым ударом больше не нужно. Если в направлении
камеры никого нет, текущая цель сохраняется и удар идёт по ней.

### Камера в бою
Камера сама плавно заходит за спину Дарси при смене цели и когда враг
обошёл сбоку (в оригинале камера в бою статична).

## Конфигурация

### .ini файл (убрано)
- Оригинальный файл конфигурации (.ini) больше не поддерживается (планируется замена на другой формат).

### Мёртвая зона стиков — настраивается, меню больше не «уезжает» от дрифта (новое)
- В `config.json` появилась секция `gamepad` с двумя мёртвыми зонами стика (доля от 0 до 1): отдельно **для игры** (движение/прицел) и отдельно **для навигации по меню**.
- Менюшная мёртвая зона по умолчанию увеличена. Раньше при дрифте стика (особенно на Xbox-геймпадах) меню могло бесконечно перелистываться само, хотя стик не трогают — теперь нет.
- Игровая мёртвая зона по умолчанию без изменений. Обе можно подстроить под свой геймпад в `config.json`.

### Лимит FPS — настраивается (новое)
- В `config.json` (секция `video`) появился ключ `fps_cap` — максимальный FPS рендера. По умолчанию **300**.
- Значение **≤ 0 = без лимита** (unlimited).
- Значения **ниже 30 поднимаются до 30**: ниже 30 fps картинка дёрганая (плюс на низком капе есть периодический рывок), поэтому диапазон <30 не даётся.
- Пока настраивается только через файл (пункта в меню ещё нет).

### Радар по умолчанию крутится от камеры (изменено)
- Поворот радара (отметки врагов и целей) по умолчанию теперь следует за камерой, а не за направлением Дарси. Позиция отметок по-прежнему относительно Дарси — меняется только ориентация.
- Поведение настраивается опцией `scanner_follows` (в меню настроек и в `config.json`); переключив её, можно вернуть поворот от Дарси.
