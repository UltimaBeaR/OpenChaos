# Геймпад доработки (часть Этапа 13)

Общие доработки геймпада (generic через SDL3 + моменты общие для всех контроллеров).

DualSense-специфика (adaptive triggers, LED, тачпад, гироскоп, haptics) → [stage13_dualsense.md](stage13_dualsense.md)
Техническая документация по DualSense API → [stage5_1_dualsense.md](stage5_1_dualsense.md)

## Настройки геймпада через UI
- В 1.0 настройки джойстика убраны из меню
- После 1.0 — полноценный UI: биндинг кнопок, чувствительность стика, деадзоны и т.д.

## Иконки кнопок в UI
- Инфраструктура `InputDeviceType` готова (определяем тип контроллера)
- Рисование соответствующих иконок (Xbox / PlayStation / клавиатура) в подсказках на экране — пока не реализовано

## Stick drift в меню
- В главном меню нет deadzone для стиков — бесконечный скролл на контроллерах с drift
- В геймплее deadzone есть, в меню — нет
- **Это баг для 1.0**, не для 13 этапа — записан в [known_issues](known_issues_and_bugs.md) — "Бесконечный скролл меню (stick drift)"
