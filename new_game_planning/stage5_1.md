# Этап 5.1 — Поддержка геймпадов (Xbox + DualSense)

**Цель:** прямая поддержка Xbox контроллера и DualSense (PS5) контроллера.

Вынесено из этапа 10 — приоритетная фича, реализуется сразу после фиксов пре-релизных багов,
до перехода на OpenGL.

## Что делаем

- **Xbox контроллер** — прямая поддержка через SDL2 GameController API (уже в зависимостях)
- **DualSense (PS5)** — прямая поддержка, включая haptics/adaptive triggers если возможно
- Переиспользуем существующее поведение PS1 контроллера из оригинала как базу для маппинга
- Маппинг кнопок настраиваемый

## Зависимости

- SDL2 уже подключен (vcpkg) — GameController API поддерживает оба контроллера
- Текущий input через DirectInput (`engine/input/joystick.*`) — нужно либо заменить на SDL2,
  либо добавить SDL2 путь параллельно
- `ui/interfac.*` — маппинг input → game actions, нужно расширить под новые контроллеры

## Что НЕ делаем на этом этапе

- Полная переработка управления (улучшения UX → этап 10)
- Замена клавиатуры/мыши на SDL2 (можно, но не обязательно)

## Референсы

- `PieroZ/MuckyFoot-UrbanChaos` коммит `57c4560` — реализация controller support (DirectInput-based). Глянуть подход при реализации нашей SDL2 версии.
- `PieroZ/MuckyFoot-UrbanChaos` коммит `1ef00b7` (Kai) — аналоговое управление ломает клавиатуру. Учесть при реализации.
