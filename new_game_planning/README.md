# Планирование новой версии Urban Chaos

## Концепция

Переписать Urban Chaos (1999) с нуля на современном стеке.
Оригинальная игра — эталон поведения и документация. Ресурсы переиспользуются.

## Цели

### Обязательно (Шаг 1 — воспроизвести оригинал)
- Тот же геймплей, та же физика, тот же ИИ
- Те же уровни, те же ресурсы
- Поддержка Windows, Linux, macOS
- Современная, читаемая кодовая база

### Позже (Шаг 2 — новый функционал)
- Поддержка геймпада/контроллера
- Возможно улучшенная графика
- Другие улучшения по необходимости

### Не нужно
- Поддержка PSX/Dreamcast
- Встроенный редактор карт
- Редактор анимаций
- Glide/software renderer
- Экспериментальный контент оригинала (мотоциклы и т.д.)

## Текущий статус

**Фаза:** Планирование архитектуры новой игры (Фаза 2)

**Следующие шаги:**
1. Финализировать выбор графического API (OpenGL 4.1 vs Vulkan)
2. Финализировать выбор звуковой библиотеки (OpenAL Soft vs miniaudio)
3. Начать реализацию (Фаза 3)
