## Управление

| Проблема | Описание | Статус |
|----------|----------|--------|
| Геймпад: R3 не работает на свежеподключённом DualSense пока не нажмёшь Enter с клавиатуры | **Репро (2026-05-20, RTA):** загрузил миссию RTA → нажал Enter (смена оружия с клавиатуры) → подключил DualSense по Bluetooth → ввёл чит на оружие и взял его → нажал **R3 на геймпаде для смены оружия — не сработало**. Затем нажал Enter с клавиатуры — оружие сменилось. После этого R3 на геймпаде **начал работать**. То есть R3 был "глухим" до тех пор, пока **один раз** не сработала смена оружия с клавиатуры. **Класс:** связан с двумя другими записями про **рассинхрон input state при подключении/отключении геймпада в середине геймплея**. Здесь конкретно: при mid-game BT-connect геймпада какой-то контекст input'а (snapshot инвентаря для `CONTROLS_rot_inventory`, либо флаг "active device", либо edge-detect состояние R3) не инициализируется. Первый `CONTROLS_new_inventory`-вызов с клавиатуры пересчитывает state и "будит" gamepad-путь. **Гипотеза по чинить:** на event подключения нового HID-устройства принудительно сбрасывать input snapshots (inventory rotation, last-edge per-button state) — аналогично disconnect-сценарию. Типичный сценарий "сел играть с клавиатуры, потом подключил геймпад", в котором первое нажатие R3 не реагирует. | Исследовать |
| Геймпад: R3 не прокручивает инвентарь при подборе оружия с открытым popup | **Репро (2026-04-15):** игрок нажимает R3 → открывается popup выбора оружия. Пока popup открыт, игрок подбирает новое оружие с земли. Новое оружие появляется в popup, но **выбрать его R3-вращением нельзя** — ротация крутит только по старому списку (до подбора). Закрыть popup и открыть заново — новое оружие выбирается нормально. **Workaround:** расфокус окна + фокус обратно снимает залипание (аналогично багу с курком DualSense). **Причина (установлена):** снапшот списка оружий в `CONTROLS_rot_inventory` не пересчитывается когда инвентарь меняется во время открытого popup, и rising-edge R3 крутит по устаревшему списку. **Фикс направление:** пересчитывать snapshot при изменении инвентаря или закрывать popup на подборе оружия. Сценарий нечастый, но при попадании выглядит как поломанная фича. | Исследовать (есть репро) |
