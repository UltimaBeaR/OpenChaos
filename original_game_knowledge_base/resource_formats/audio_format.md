# Форматы аудио — Urban Chaos

**Расположение:** `original_game/fallen/Debug/data/sfx/`, `original_game/fallen/Debug/talk2/`
**Движок:** Miles Sound System (MSS32) — `mss32.dll` / `mss32.lib`
**Заголовки:** `fallen/Headers/Sound.h`, `fallen/Source/Sound.cpp`, `fallen/Source/music.cpp`

---

## Звуковые эффекты (SFX)

### Формат файлов
- **WAV** — стандартный формат, PCM 16-bit, моно или стерео
- Частота дискретизации: обычно 22050 Hz или 44100 Hz

### Расположение
```
data/sfx/1622/
├── Ambience/      — фоновые звуки (ветер, птицы, городской шум)
├── Footsteps/     — шаги по разным поверхностям
├── General/       — общие SFX (выстрелы, удары, взрывы и т.д.)
├── music01/       — музыкальные треки, вариант 1
├── music02/       — вариант 2
│   ...
└── music08/       — вариант 8
```

### Внутри каждой music0N папки:
```
musik0N/
├── Crawl/         — тихая/скрытная музыка
├── Driving1/      — езда (вариант 1)
├── Driving2/      — езда (вариант 2)
├── DrivingStart/  — старт двигателя
├── Fight1/        — бой (вариант 1)
├── Fight2/        — бой (вариант 2)
├── Sprint1/       — спринт/погоня (вариант 1)
└── Sprint2/       — спринт/погоня (вариант 2)
```

## Диалоги

### Формат
- **WAV** — PCM 16-bit, моно

### Расположение и именование

talk2/

## Музыкальная система (14 режимов)

Музыка переключается динамически в зависимости от действий игрока.
Подробнее — см. [subsystems/audio.md](../subsystems/audio.md).

Ключевые функции (`music.cpp`):
```c
MUSIC_set_mode(mode)    // установить режим (0–13)
MUSIC_update()          // вызывается каждый тик, плавные переходы
```

---

## Miles Sound System (MSS32)

MSS32 — проприетарная библиотека (RAD Game Tools).
- 3D позиционный звук (HRTF)
- Потоковое воспроизведение музыки
- Смешивание множества каналов

---

## Звуковые ID (sound_id.h)

Все звуки адресуются по числовым ID:
```c
// sound_id.h — некоторые примеры
#define S_ZIPWIRE   113    // звук скольжения по тросу
// ... (сотни других ID)
```
