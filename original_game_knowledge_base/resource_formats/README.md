# Форматы ресурсов

## Статус: 🔲 Требует детального анализа

Ресурсы игры находятся в `original_game/fallen/Debug/`.
Это Steam-версия ресурсов — рабочая.

Загрузчик ресурсов будет перенесён 1:1 в новую версию.
Понимание форматов критично для написания загрузчика.

## Структура папки ресурсов

Ресурсы находятся в **подпапках** `original_game/fallen/Debug/`.
Корень Debug/ также содержит артефакты сборки (.obj, .exe, .dll и т.д.) — они не нужны.

```
Debug/
├── Meshes/      — 3D меши
├── Textures/    — текстуры
├── clumps/      — clump-файлы (модели персонажей и объектов)
├── data/        — игровые данные (миссии, настройки и т.д.)
├── levels/      — файлы уровней/карт
├── talk2/       — диалоги
├── text/        — текстовые строки (локализация)
├── bink/        — Bink-видео (катсцены)
├── stars/       — текстуры звёздного неба
├── server/      — (уточнить назначение)
└── config.ini   — конфигурационный файл игры
```
