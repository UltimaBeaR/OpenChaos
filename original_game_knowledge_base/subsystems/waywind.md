# WayWind — редакторный UI компонент

**Файл:** `fallen/Source/WayWind.cpp` (353 строки)

## ⚠️ Не является игровой системой

WayWind — **инструмент редактора уровней** (`GEdit.exe`), не часть игрового рантайма.
В `fallen/Source/` нет ни одного `#include "waywind.h"`.

---

## Что делает

Создаёт 15 цветных кнопок в UI редактора (диалог waypoint'ов):

```c
#define WAY_COLOURS 15

UBYTE button_colours[15][3] = {
    { 0, 0, 0 },            // 0: Black
    { 255, 255, 255 },      // 1: White
    { 255, 0, 0 },          // 2: Red
    { 255, 255, 0 },        // 3: Yellow
    { 0, 255, 0 },          // 4: Green
    { 0, 255, 255 },        // 5: Cyan
    { 0, 0, 255 },          // 6: Blue
    { 255, 0, 255 },        // 7: Magenta
    // ... ещё 7 кастомных цветов
};

BOOL init_wwind(void);  // Регистрирует 15 WNDCLASSEX (Windows button классов)
void fini_wwind(void);  // Отменяет регистрацию при выходе
```

Эти 15 цветов = `pcom_colour` (0–15) — цвета группировок NPC в редакторе. Кнопки позволяют назначить цвет (группировку) waypoint'ам при создании NPC.

---

## Связь с игрой

WayWind сам — только UI. Цвета из него идут в `pcom_colour` поле NPC-персонажей (документировано в [ai.md](ai.md) раздел "Групповое поведение").

Функциональность принадлежит редактору — вне скоупа анализа, см. [analysis_scope.md](../analysis_scope.md).
